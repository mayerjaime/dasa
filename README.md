# HTTParty | Cucumber

Repositório do Teste Pratico - Automacao API - DASA

## Bundler
Instale o Bundler:

```ruby
$ gem install bundler
```

## Projeto

>Faça o Clone do projeto
>Execute o comando  para instalar as dependências
```ruby
$ bundle install
```
>Rode o projeto usando o comando

```ruby
$ cucumber
```
